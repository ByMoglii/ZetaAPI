# ZetaAPI

Official developer API for Zeta Anticheat.

Zeta is an advanced Minecraft anticheat with a modular detection system.

Main plugin:
https://www.spigotmc.org/resources/zeta-mineguard-%E2%80%93-anticheat-system.124786/

## Installation

Put `ZetaAPI.jar` into your server's `plugins` folder.

Do not shade ZetaAPI into your plugin.
Use `provided` for Maven or `compileOnly` for Gradle.
